# MDLBSG Compressor v0.1.0

Released as the first public version of MDLBSG Compressor.

Internal application build: **App 91**

## Included

- macOS compression application
- File and folder compression
- MDLBSG archive restoration
- Single live work queue
- Turbo compression mode
- Optional plaintext cache, disabled by default
- Installer and release verification scripts
- Compression, queue, interface, and installer source files

## Primary Testing

Development and testing were primarily performed on:

- macOS
- Apple Silicon
- The full 1,000,000,000-byte `enwik9` dataset
- Regular files and folders
- Compression followed by exact restoration
- Multiple submissions through the live queue

## Current Scope

This is an experimental early public release.

The Turbo mode is the recommended mode for this version because the other included modes have not received the same level of testing.

Keep original copies of important data until the resulting archive has been restored and checked.
